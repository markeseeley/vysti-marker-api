How Vysti works

Three steps. Consistent results. No guesswork.

1

Upload

Drop in an essay or a full class set — .docx files, any length.

2

Analyse

Vysti applies 100+ deterministic rules across four scoring dimensions.

3

Improve

Students revise with targeted feedback. Teachers return scored, annotated essays.

Four dimensions of writing quality

Every essay is scored on the same four metrics — rules, not opinions.

Power
Verb strength and impact
Analysis
Analytical depth and evidence
Cohesion
Flow and logical connection
Precision
Word choice accuracy
For Students

Revise with confidence

Upload your essay and get instant, rule-based feedback on every paragraph. Vysti shows you exactly what to fix and why — then lets you edit and recheck until you're satisfied.

  • Four performance meters based on concrete rules, not opinions
  • Edit in-browser and recheck — same rules, updated score
  • Every issue links to the exact sentence with the rule behind it
  • Download your polished draft when you're done
For Teachers

Mark an entire class in minutes

Upload a class set and get every essay scored against identical standards. Vysti handles the consistency so you can focus on the human touch.

  • Batch-mark an entire class in one upload
  • Annotation toolkit: highlights, strikethroughs, comments
  • Deterministic score card — same essay, same result, every time
  • Download marked essays ready to return to students
